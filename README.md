# Estructura de Datos en C (TDAs)

Este repositorio contiene implementaciones de Tipos de Datos Abstractos (TDAs) desarrolladas en lenguaje C, como parte de un curso universitario de algoritmos y estructuras de datos.

El objetivo del proyecto es diseñar, implementar y utilizar estructuras de datos fundamentales, prestando especial atención al manejo de memoria, la modularidad y la claridad de las interfaces.

Cada estructura se encuentra organizada en su propio directorio, con documentación específica.

---

## Estructuras implementadas

### Lista
Implementación de una lista enlazada, utilizada como estructura base para el almacenamiento y recorrido de datos.

Conceptos trabajados:

- Punteros y estructuras dinámicas
- Manejo explícito de memoria
- Encapsulamiento mediante interfaces

Ver detalles en: `LISTA/README.md`

### Árbol Binario de Búsqueda (ABB)
Implementación de un árbol binario de búsqueda, orientado a operaciones eficientes de inserción, búsqueda y eliminación.

Conceptos trabajados:

- Estructuras jerárquicas
- Recorridos de árboles
- Comparación y ordenamiento de datos
- Recursión

Ver detalles en: `ABB/README.md`

### Tabla Hash
Implementación de una tabla hash, enfocada en acceso eficiente a datos mediante claves.

Conceptos trabajados:

- Funciones hash
- Resolución de colisiones
- Estructuras auxiliares
- Análisis de eficiencia

Ver detalles en: `hash/README.md`

---

## Objetivos alcanzados

- Comprender y aplicar estructuras de datos fundamentales
- Diseñar TDAs con interfaces claras y bien definidas
- Gestionar memoria de forma segura y eficiente
- Escribir código modular y reutilizable en C
- Resolver problemas algorítmicos en C
- Analizar la complejidad temporal y espacial (Big-O) de las operaciones
- Validar el correcto funcionamiento de las estructuras mediante testing
